Application name: HiMum
Author: Senthilkumar R

Steps:
1. Install NodeJs (v12.18.3) globally.
2. Install NPM (6.14.6) globally.
3. Open the root folder of the project and run "npm install" or "npm i".
4. The command at step 3 will install the node modules in the root folder.
5. Open the terminal from the root folder and run "npm start".
6. It will build the application and launch the browser.

Other Information:
Gulp is used to build the application hence it will automate the build process. Which means that, any changes that are done in src folder or *.html files. It will build the application and launch the browser.
